# kid3 で使用されているカバーアートの種類

> 出典: https://invent.kde.org/multimedia/kid3/-/blob/master/src/core/tags/pictureframe.cpp

|  ID  |  説明  |
|-----:|--------|
|    0 | Other |
|    1 | 32x32 pixels PNG file icon |
|    2 | Other file icon |
|    3 | Cover (front) |
|    4 | Cover (back) |
|    5 | Leaflet page |
|    6 | Media |
|    7 | Lead artist/lead performer/soloist |
|    8 | Artist/performer |
|    9 | Conductor |
|   10 | Band/Orchestra |
|   11 | Composer |
|   12 | Lyricist/text writer |
|   13 | Recording Location |
|   14 | During recording |
|   15 | During performance |
|   16 | Movie/video screen capture |
|   17 | A bright coloured fish |
|   18 | Illustration |
|   19 | Band/artist logotype |
|   20 | Publisher/Studio logotype |

kid3-cli でミュージックファイルにカバーアートの種類とともにカバーアート画像を付加したい場合には以下のコマンドを使用する。
```
kid3-cli -c "set picture:\"<カバーアート画像のパス名>\" \"\"" -c "set picture.picturetype <カバーアートの種類を示すID>" <設定対象ミュージックファイルパス名対象>
```

サンプルは以下の通り。

```
kid3-cli -c "set picture:\"cover.jpg\" \"\"" -c "set picture.picturetype 4" sample.mp3

```

複数 (n個) のカバーアートを付与したい場合は以下のコマンドを使用する。
```
kid3-cli \
-c "set picture[0]:\"<1番目のカバーアート画像のパス名>\" \"\"" -c "set picture[0].picturetype <1番目のカバーアートの種類を示すID>" \
-c "set picture[1]:\"<2番目のカバーアート画像のパス名>\" \"\"" -c "set picture[1].picturetype <2番目のカバーアートの種類を示すID>" \
...
-c "set picture[<n-1>]:\"<n番目のカバーアート画像のパス名>\" \"\"" -c "set picture[<n-1>].picturetype <n番目のカバーアートの種類を示すID>" \
<設定対象ミュージックファイルパス名対象>
```
